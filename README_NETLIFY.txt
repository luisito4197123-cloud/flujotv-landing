Cómo publicar en Netlify (multipágina)

1) Ve a app.netlify.com → Sites → Add new site → Deploy manually.
2) Arrastra el ZIP completo. Netlify detecta index.html automáticamente.
3) La navegación es real: /index.html, /servicios.html, /contacto.html.
4) Archivo _redirects incluido para que /servicios y /contacto funcionen.
5) Usa la URL de Netlify en tu apelación (APELACIÓN_GOOGLE_ADS.txt).

Consejo: usa esta web solo para anuncios. No agregues textos de 'canales/TV'.
